# HTTP Status Display Server

Простой HTTP-сервер на Python для визуализации состояния системы через веб-интерфейс.

## Возможности

- Отображение статуса в реальном времени через браузер
- Управление состоянием через URL-параметры
- Цветовая индикация: зелёный (активно), красный (отключено), серый (не определено)
- Работает без установки дополнительных пакетов

## Системные требования

- Python 3.6 или выше
- Любой современный веб-браузер

## Установка и запуск

Сохраните код сервера в файл `server.py`, затем выполните в терминале:

```bash
python3 server.py
```

Сервер запустится на порту 8000 и будет ожидать подключения.

## Доступ к интерфейсу

После запуска откройте в браузере:

```
http://localhost:8000
```

## Переключение состояний

Состояние системы задаётся через query-параметр `status` в URL:

**Активировать систему:**
```
http://localhost:8000/?status=on
```

**Деактивировать систему:**
```
http://localhost:8000/?status=off
```

**Сбросить состояние:**
```
http://localhost:8000
```

## Визуальная индикация

| Состояние | Цвет фона | Отображаемый текст |
|-----------|-----------|-------------------|
| Активно | Зелёный (#10b981) | ON |
| Отключено | Красный (#ef4444) | OFF |
| Не определено | Серый (#94a3b8) | ? |

## Остановка сервера

Для остановки нажмите `Ctrl+C` в терминале, где запущен сервер.

## Особенности реализации

- Используется только стандартная библиотека Python (`http.server`)
- Минимальный HTML с inline-стилями
- Автоматическая обработка некорректных параметров
- Отключено логирование запросов для чистого вывода
