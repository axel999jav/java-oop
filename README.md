Назначение
Сайт отображает статус системы (On / Off) в виде цветной плашки.
Статус меняется через адресную строку — без входа в админку и без сложных настроек. Как изменить статус
Шаг 1. Откройте адрес сайта
1
Шаг 2. Добавьте параметр статуса в конец адреса
Что нужно
Как написать в адресной строке
🔹 Включить
https://site.com/?status=on
🔹 Выключить
https://site.com/?status=off
Шаг 3. Нажмите Enter
Страница перезагрузится, и плашка изменит цвет и текст:
🟢 On — зелёная, с пульсирующей точкой
🔴 Off — красная, статичная
